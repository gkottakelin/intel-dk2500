# RGB 视觉闭环运动规划开发方案

## 1. 项目目标

本阶段目标是在主机上接入云端 AI，通过彩色图像实时理解任务、检测目标，并利用视觉闭环控制机械臂完成桌面物体抓取、放置和叠放任务。

典型任务示例：

- “夹取红色的物块放到旁边的蓝色框子里”
- “将红色的物块叠放到黄色物块上”

本方案只使用 RGB 彩色图像。

## 2. 已确认条件

### 2.1 机械臂 home 位置

```text
J1 = 500
J2 = 550
J3 = 550
J4 = 900
J5 = 500
J6 = 360
```

其中 J6=360 认为是松开状态。

### 2.2 控制模式

```text
J1-J5: servo 模式
J6: motor 模式
```

### 2.3 J6 夹爪控制策略

```text
松开位置: 360
夹取速度: 100
张开速度: -100
夹紧速度: 300
夹取判定: 夹取过程中连续 2 秒位置不变，认为已经夹到物体
```

J6 的真实运动位置范围为 0-1000。

```text
0: 完全张开
700: 几何闭合
700-1000: 几何上保持闭合，只增加夹持力
```

### 2.4 相机条件

- 只读取彩色图像。
- 相机安装在 J5/J6 附近，会随 J5 旋转。
- 相机可以看到夹爪。
- 当前夹爪已贴紫红色大区域和橙色爪尖标记。

### 2.5 工作空间条件

- 抓取目标都位于桌面上。
- 桌面高度定义为 z=0。
- 每次任务完成后回 home。
- 安全边界使用当前关节限位配置。

## 3. 总体架构

系统分为五层：

```text
自然语言任务输入
        ↓
云端大模型任务解析与目标检测
        ↓
本地 RGB 视觉检测
        ↓
视觉闭环运动规划
        ↓
舵机执行与安全监控
```

### 3.1 云端大模型层

云端大模型负责：

- 理解自然语言任务。
- 从 RGB 图像中检测目标物体。
- 输出结构化任务 JSON。

云端大模型不直接控制舵机。

### 3.2 本地视觉层

本地视觉负责实时闭环，主要检测：

- 左爪橙色端点。
- 右爪橙色端点。
- 夹爪中心。
- 夹爪开口方向。
- 目标中心。
- 放置区域中心。

本地视觉必须低延迟运行，不依赖云端每一帧返回。

### 3.3 视觉闭环规划层

闭环控制不直接相信舵机开环精度，而是使用图像误差反复修正：

```text
拍图
检测夹爪中心和目标中心
计算图像误差
生成 J1-J5 小步修正
执行 servo
再次拍图
```

### 3.4 舵机执行层

J1-J5 执行小步 servo 修正。

J6 使用 motor 模式执行抓取和释放。

## 6. 夹爪视觉检测方案

### 6.1 标记颜色使用

当前夹爪标记：

- 紫红色区域：用于粗定位夹爪主体。
- 橙色爪尖：用于精确定位夹爪末端。

后续闭环优先使用橙色爪尖。

### 6.2 夹爪端点定义

```text
left_tip = 左橙色端点中心
right_tip = 右橙色端点中心
gripper_center = (left_tip + right_tip) / 2
gripper_width_px = distance(left_tip, right_tip)
gripper_axis = right_tip - left_tip
```

### 6.3 夹爪检测流程

优先检测橙色爪尖：

```text
RGB 图像
  ↓
HSV 转换
  ↓
橙色 mask
  ↓
形态学去噪
  ↓
连通域分析
  ↓
选择两个最稳定区域
  ↓
得到 left_tip / right_tip
```

如果橙色检测失败，则使用紫红色区域辅助恢复左右爪大致位置。

## 7. 启动标定流程

### 7.1 回 home

```text
J1 = 500
J2 = 550
J3 = 550
J4 = 900
J5 = 500
J6 = 360
```

### 7.2 颜色采样

允许每次启动时做一次颜色采样。

采样内容：

- 左橙色爪尖。
- 右橙色爪尖。
- 可选：紫红色爪面。

保存为配置：

```yaml
gripper_tip_color:
  hsv_lower: [...]
  hsv_upper: [...]

gripper_body_color:
  hsv_lower: [...]
  hsv_upper: [...]
```

### 7.3 相机内参标定

使用棋盘格或 Charuco 标定板。

采集 20-40 张不同角度图像，使用 OpenCV 标定：

```text
camera_matrix
dist_coeffs
```

后续每帧先进行去畸变。

### 7.4 视觉伺服 Jacobian 标定

在 home 附近，对 J1-J5 分别做小步扰动：

```text
J1 +10 / -10
J2 +10 / -10
J3 +10 / -10
J4 +10 / -10
J5 +10 / -10
```

每次扰动后记录夹爪中心像素变化：

```text
delta_joint -> delta_pixel_error
```

保存为：

```yaml
visual_servo_jacobian:
  joints: [J1, J2, J3, J4, J5]
  pixel_per_position_unit:
    J1: [du, dv]
    J2: [du, dv]
    J3: [du, dv]
    J4: [du, dv]
    J5: [du, dv]
```

运行时计算：

```text
delta_joint = pseudo_inverse(J) * pixel_error
```

并限制每次单步大小。

## 8. 视觉闭环运动规划

### 8.1 基本误差

```text
error = target_center - gripper_center
```

### 8.2 闭环循环

```text
1. 拍图
2. 检测夹爪中心
3. 检测目标中心
4. 计算 error
5. 根据视觉 Jacobian 生成 J1-J5 小步修正
6. 执行 servo
7. 重复直到 error 小于阈值
```

### 8.3 收敛条件

```text
abs(du) < pixel_threshold_u
abs(dv) < pixel_threshold_v
目标中心位于两爪之间
```

### 8.4 单步限制

J1-J5 每次闭环只允许小步移动。

建议初始值：

```text
max_delta_per_step = 5-20 raw units
```

最终值需要通过实机测试确定。

## 9. 抓取流程

```text
1. 回 home
2. J6 张开到 360
3. 云端大模型识别 source 目标
4. 本地视觉检测夹爪中心和目标中心
5. 视觉闭环让目标进入两爪中间
6. J6 motor speed = 100 开始夹取
7. 持续读取 J6 位置
8. 如果 2 秒位置不变，判定夹到目标
9. J6 motor speed = 300 短暂加力
10. 停止 J6
```

## 10. 放置流程

```text
1. 云端大模型识别 target
2. 本地视觉检测 target 中心
3. 视觉闭环让 target 位于夹爪中心下方
4. 接近过程中继续视觉闭环
5. J6 motor speed = -100 松开
6. 直到 J6 position <= 360
7. 停止 J6
8. 回 home
```

## 11. 叠放流程

```text
1. 识别 source 物体
2. 抓取 source
3. 识别 target 物体
4. 视觉闭环对准 target 中心
5. 通过图像观察 source 与 target 的接触或遮挡变化
6. 到达释放条件后松开
7. 回 home
```

物块尺寸、高度和释放高度不预先写死，依赖视觉闭环逐步接近和判断。

## 12. 安全策略

### 12.1 关节限位

J1-J5 严格使用当前 YAML 中的限位。

### 12.2 桌面安全

桌面高度定义为：

```text
z = 0
```

规划不得低于桌面。

### 12.3 视觉丢失处理

如果出现以下情况：

- 夹爪标记丢失。
- 目标丢失。
- 云端检测置信度不足。
- 闭环长期不收敛。
- 舵机位置异常。

执行：

```text
停止 J1-J5
停止 J6
根据状态选择回 home 或等待人工确认
```

### 12.4 任务结束

每次任务完成后统一回 home。

## 13. 开发里程碑

### 阶段 1：夹爪颜色检测

- 实现橙色爪尖采样。
- 实现紫红爪面采样。
- 实现 HSV 阈值保存和加载。
- 实现左右爪尖检测。

### 阶段 2：云端大模型接口

- 定义图像上传接口。
- 定义 JSON 返回格式。
- 实现自然语言任务解析。
- 实现目标 bbox / center 返回。

### 阶段 3：视觉误差计算

- 计算夹爪中心。
- 计算目标中心。
- 计算 error。
- 可视化检测结果。

### 阶段 4：视觉 Jacobian 标定

- 自动执行 J1-J5 小步扰动。
- 记录像素变化。
- 保存局部 Jacobian。
- 实现 pseudo-inverse 修正。

### 阶段 5：单目标闭环对准

- 给定目标中心。
- 控制夹爪中心对准目标。
- 实机测试收敛速度和稳定性。

### 阶段 6：单目标抓取

- 对准目标。
- J6 motor 夹取。
- 2 秒位置不变判定夹取。
- 夹紧后停止。

### 阶段 7：放置任务

- 识别放置区域。
- 视觉闭环对准。
- 松开。
- 回 home。

### 阶段 8：叠放任务

- 识别 source 和 target。
- 抓取 source。
- 对准 target。
- 视觉判断释放。
- 回 home。

### 阶段 9：安全恢复和日志

- 目标丢失恢复。
- 夹爪丢失恢复。
- 任务失败回 home。
- 保存任务日志和图像快照。
