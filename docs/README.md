## 当前重要结论

- 当前任务只使用 RGB 彩色图。
- 相机安装在 J5/J6 附近，会随 J5 转动，并且能看到夹爪。
- 夹爪末端使用橙色标记作为主定位特征，紫红色大区域作为辅助识别特征。
- J1-J5 使用 servo 模式做视觉闭环运动，J6 使用 motor 模式完成夹取和释放。
- 每次任务完成后回 home：`J1=500, J2=550, J3=550, J4=900, J5=500, J6=360`。
