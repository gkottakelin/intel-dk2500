# 项目进度总结

> 项目目标：构建一个基于 RGB 视觉、AI 自然语言任务理解和总线舵机控制的机械臂平台，实现物块识别、视觉闭环定位、运动规划和机械臂执行。

## 1. 当前状态

当前项目主线已经统一为：

```text
RGB 彩色图
-> 云端大模型任务解析
-> 本地目标/夹爪标记检测
-> 视觉闭环控制 J1-J5
-> J6 motor 模式夹取/释放
```

已确认条件：

| 项目 | 状态 |
|---|---|
| home | `J1=485, J2=478, J3=641, J4=890, J5=500`；J6 保持当前状态，home 时不发 J6 信号 |
| 相机 | 安装在 J5/J6 附近，会随 J5 转动，并能看到夹爪 |
| 视觉输入 | 只使用 RGB 彩色图像 |
| 夹爪标记 | 橙色爪尖用于精确定位，紫红色区域作为辅助识别 |
| J1-J5 | servo 模式进行闭环运动 |
| J6 | motor 模式进行夹取和释放 |
| 安全 | 使用现有关节限位，桌面高度为 `z=0`，每次任务结束回 home |

## 4. 下一阶段建议

### 阶段 A：RGB 图像和夹爪标记稳定检测

1. 固定相机分辨率和曝光。
2. 使用 HSV 阈值检测橙色爪尖。
3. 使用紫红色区域作为夹爪辅助检测。
4. 建立目标检测结果格式。
5. 保存样本图、检测框、夹爪端点和失败案例。

### 阶段 B：视觉闭环标定

1. 在 home 附近记录每个关节小步运动对图像中夹爪/目标位置的影响。
2. 建立视觉伺服雅可比或经验增量表。
3. 设定每轮闭环最大步长和最小误差阈值。
4. 加入丢失目标、夹爪不可见、画面过暗等异常退出条件。

### 阶段 C：抓取闭环

1. 从 home 进入观察姿态。
2. 视觉闭环让目标位于两爪中线。
3. 夹取时 J6 速度设为 `100`。
4. 夹取过程中持续读取 J6 位置。
5. 连续 `2s` 位置不变后认为夹到物体。
6. 将 J6 速度提高到 `300`，确保夹紧。
7. 抬起后视觉确认目标是否随夹爪移动。

### 阶段 D：放置和叠放闭环

1. 识别放置目标，例如蓝色框、黄色物块或指定区域。
2. 使用视觉闭环对准放置位置。
3. 释放高度依赖视觉闭环逐步确认，不写死固定高度。
4. J6 速度 `-100` 张开释放。
5. 释放后视觉确认目标已经离开夹爪。
6. 每次任务完成后回 home。
