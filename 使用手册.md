## 2. 开发环境

| 环境 | 用途 |
|---|---|
| Windows + Python | 舵机串口调试、RGB 图像处理和云端大模型接入开发 |
| WSL2 Ubuntu 22.04 + ROS 2 Humble | 运行 `jetarm_description`、RViz 显示、后续视觉闭环和控制节点开发 |

Windows 项目目录：

```text
D:\jetarm\project
```

WSL 中访问同一目录：

```bash
/mnt/d/jetarm/project
```

ROS2 工作区建议放在 Ubuntu 文件系统中：

```bash
~/ros2_ws
~/ros2_ws/src
```

## 3. ROS2 机械臂显示

```bash
source /opt/ros/humble/setup.bash
mkdir -p ~/ros2_ws/src
cp -r /mnt/d/jetarm/project/ros2_ws/src/jetarm_description ~/ros2_ws/src/
chmod +x ~/ros2_ws/src/jetarm_description/scripts/joint_state_mapper.py
cd ~/ros2_ws
colcon build --packages-select jetarm_description
source install/setup.bash
ros2 launch jetarm_description display.launch.py
```

如果缺少 GUI 包：

```bash
sudo apt update
sudo apt install -y ros-humble-joint-state-publisher-gui
```

## 4. 当前 J6 夹爪约定

J6 真实舵机位置范围为 `0..1000`：

| J6 位置 | 含义 |
|---|---|
| `0` | 两根爪子完全展开，成一条直线 |
| `700` | 两根爪子几何闭合，沿 J5 到 TCP 方向重合 |
| `700..1000` | 几何上保持闭合，只增加夹持力 |

任务执行时采用 motor 模式：

| 动作 | 参数 |
|---|---|
| 松开状态 | `J6=360` 认为已松开 |
| 夹取 | 速度 `100`，一边运动一边读取位置 |
| 夹到判定 | 连续 `2s` 位置不变 |
| 夹紧 | 判定夹到后速度增加到 `300` |
| 张开 | 速度 `-100` |

## 5. 总线舵机调试

下面命令默认从 Windows PowerShell 的 `D:\jetarm` 目录运行。

读取状态：

```powershell
python project\src\bus_servo.py status COM3 1
```

servo 位置模式：

```powershell
python project\src\bus_servo.py servo COM3 1 500 1000
```

motor 速度模式：

```powershell
python project\src\bus_servo.py motor COM3 1 100
```

## 6. 测试

从 `D:\jetarm` 运行：

```powershell
python -m unittest discover -s project\tests
```

当前重点测试：

| 测试 | 作用 |
|---|---|
| `tests/test_bus_servo.py` | 验证总线舵机协议组帧、校验和状态解析 |
| `tests/test_urdf_description.py` | 验证 ROS2 机械臂描述、J6 夹爪和 launch 参数 |

## 7. 当前开发顺序

1. 固定 RGB 相机安装位置，保证能看到夹爪和桌面目标。
2. 采集 RGB 样本，标注目标和夹爪橙色爪尖。
3. 实现本地夹爪标记检测。
4. 接入云端大模型，返回结构化任务 JSON。
5. 标定像素误差到 J1-J5 小步动作的响应关系。
6. 实现视觉闭环抓取。
7. 实现放置、叠放和任务完成后回 home。
## 8. 可视化操作终端

操作终端会直接通过串口控制舵机，默认读取 `config/joint_servo_map.yaml` 中的 `default_com_port` 和 `baudrate`。

```powershell
python project\src\operation_terminal.py --com-port COM15
```

无硬件预览界面：

```powershell
python project\src\operation_terminal.py --dry-run
```

| 控件 | 舵机行为 |
|---|---|
| 上 / 下 | 抓取点按 `5 cm/s` 上升或下降，鼠标松开停止继续小步下发 |
| 前后左右摇杆 | 水平 TCP 速度控制，拖动幅度越大速度越快，松开自动回中 |
| J5 逆时针 / 顺时针 | J5 motor 速度 `-100 / +100`，鼠标松开速度置 `0` |
| J6 松 / 闭 | J6 motor 速度 `-100 / +100`，鼠标松开速度置 `0` |
| J6 抓紧 | 状态开关；红色时 J6 速度 `300` 并忽略松/闭按钮，再按变绿色并停止 J6 |
| 全部停止 | 水平/上下速度清零，J5/J6 速度置 `0` |
