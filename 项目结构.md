# 项目文件夹结构

以下树只列出有意义的源码、配置和文档；`.git/`、`__pycache__/`、临时 Word 锁文件以及批量渲染页未逐一展开。

```text
project/
├─ README.md                         项目总入口和当前技术主线
├─ 使用手册.md                        常用运行入口
├─ 环境重建说明.md                    简版环境恢复说明
├─ 项目方法总结.md                    已采用的技术方法
├─ 项目进度总结.md                    已完成项和后续工作
├─ GitHub仓库准备说明.md               仓库整理和提交说明
├─ requirements.txt                 Windows 主环境依赖
├─ requirements-venv-gemini.txt     旧 Gemini 环境快照，仅供参考
├─ config/
│  ├─ joint_servo_map.yaml          关节、舵机、限位、Home 和几何参数
│  └─ camera_structure.yaml         相机安装、内外参和标定占位配置
├─ data/
│  └─ rgbd_samples/                 RGB-D 样本和调试图像输出目录
├─ docs/
│  ├─ README.md                      文档索引
│  ├─ 总线舵机Python控制接口.md         串口协议和 API 用法
│  ├─ 方案1_视觉闭环运动规划开发方案.md
│  ├─ 方案2_云端多模态大模型机械臂控制方案.md
│  └─ 方案3_固定物块尺寸单目测距机械臂规划方案.md
├─ src/
│  ├─ arm_model.py                  正运动学、位置/角度转换、雅可比
│  ├─ bus_servo.py                  总线舵机底层协议
│  ├─ joint_controller.py           配置驱动的关节命令规划与执行
│  ├─ operation_terminal.py         Windows 图形化操作终端
│  ├─ servo_limit_tester.py         舵机限位/堵转稳定性测试
│  └─ gemini_windows/               Windows Gemini 相机实验代码
├─ tests/                           核心代码和 URDF 单元测试
├─ ros2_ws/
│  └─ src/jetarm_description/       ROS 2 描述包
│     ├─ launch/                    ROS 2 启动文件
│     ├─ scripts/                   J6 状态映射节点
│     ├─ urdf/                      实际模型和 GUI 输入模型
│     └─ rviz/                      RViz 显示配置
├─ ubuntu22_04_operation_terminal/  可独立复制的 Ubuntu 操作终端
├─ ubuntu22_04_gemini_camera/       可独立复制的 Ubuntu Gemini RGB-D 查看器
│  ├─ gemini_camera.py              设备选择、彩色/深度显示、点击测距和快照
│  ├─ orbbec_native.py              Linux OrbbecSDK 1.5.7 ctypes 最小封装
│  ├─ config/camera.json            相机序列号、深度显示和输出配置
│  ├─ sdk/                          Gemini 配置、udev 规则和 x86_64 动态库
│  └─ setup.sh / run.sh             Ubuntu 环境创建和启动脚本
├─ report_build/                    报告、图表生成和 Word 审核脚本
├─ deliverables/                    DOC/DOCX/PDF 报告及最终插图
├─ 项目规划/                         总体架构、开发计划和资料索引
└─ launch/                          预留的顶层 ROS 2 启动目录，目前为空
```
