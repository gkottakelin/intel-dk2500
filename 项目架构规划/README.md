## 项目目标

在 Intel DK2500 上运行 ROS2、视觉识别、深度测距、自然语言任务解析和运动规划。Gemini 深度相机提供 RGB-D 感知，AI 模块把自然语言转为结构化任务，运动规划模块生成机械臂轨迹，再经 ROS2 舵机控制链路和总线舵机通信协议控制机械臂完成抓取、移动、放置等动作。
