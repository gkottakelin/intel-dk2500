## 项目目标

在主机或 Intel DK2500 上运行 ROS2、RGB 视觉识别、自然语言任务解析和视觉闭环运动规划。AI 模块把自然语言转为结构化任务，本地视觉模块读取彩色图并检测目标和夹爪标记，运动规划模块通过 J1-J5 servo 闭环调整位置，再经 J6 motor 模式完成抓取、移动、放置和叠放等动作。
